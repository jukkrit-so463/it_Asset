# คู่มือการใช้งานระบบ IT Asset Management

## ภาพรวมระบบ

ระบบจัดการทรัพย์สินไอทีที่พัฒนาด้วย React + TypeScript และ Node.js + MySQL โดยมีฟีเจอร์หลักดังนี้:

### 🔐 ระบบ Authentication
- JWT Token Authentication
- Role-based Access Control (Admin/User)
- Password Hashing ด้วย bcryptjs
- Session Management

### 💻 จัดการครุภัณฑ์ (Assets)
- เพิ่ม/แก้ไข/ลบครุภัณฑ์
- จัดการ Service Tag และ Serial Number
- กำหนดผู้รับผิดชอบและสถานที่
- ติดตามสถานะการใช้งาน
- จัดการ IP Address

### 👥 จัดการผู้ใช้งาน (Users)
- เพิ่ม/แก้ไข/ลบผู้ใช้งาน
- กำหนดบทบาทและสิทธิ์
- ติดตามสถานะการใช้งาน
- จัดการข้อมูลส่วนตัว

### 📍 จัดการสถานที่ (Locations)
- เพิ่ม/แก้ไข/ลบสถานที่
- จัดกลุ่มตามฝ่ายและแผนก
- ติดตามครุภัณฑ์ในแต่ละสถานที่

### 📊 Dashboard และ Reports
- สถิติภาพรวมระบบ
- กราฟแสดงข้อมูลครุภัณฑ์
- รายงานการใช้งาน
- การวิเคราะห์ข้อมูล

## การใช้งานระบบ

### 1. การเข้าสู่ระบบ

1. เปิดเบราว์เซอร์ไปที่ `http://localhost:5173`
2. กรอก Username และ Password
3. เลือก "จดจำการเข้าสู่ระบบ" (ถ้าต้องการ)
4. คลิก "เข้าสู่ระบบ"

### 2. Dashboard

Dashboard แสดงภาพรวมของระบบ:

#### สถิติหลัก
- **ครุภัณฑ์ทั้งหมด**: จำนวนครุภัณฑ์ทั้งหมดในระบบ
- **ใช้งานปกติ**: ครุภัณฑ์ที่ใช้งานได้ปกติ
- **กำลังซ่อม**: ครุภัณฑ์ที่อยู่ระหว่างการซ่อม
- **ชำรุด**: ครุภัณฑ์ที่ชำรุดหรือปลดระวาง

#### กราฟและชาร์ต
- **ครุภัณฑ์แยกตามประเภท**: แสดงจำนวนครุภัณฑ์แต่ละประเภท
- **มูลค่าครุภัณฑ์รายเดือน**: แสดงมูลค่ารวมรายเดือน

#### การดำเนินการด่วน
- **ตั้งค่าระบบ**: เข้าถึงการตั้งค่าทั่วไป
- **Backup Database**: สำรองข้อมูลฐานข้อมูล
- **Login Logs**: ดูประวัติการเข้าใช้งาน
- **การแจ้งเตือน**: ตั้งค่าการแจ้งเตือนระบบ

### 3. จัดการครุภัณฑ์

#### ดูรายการครุภัณฑ์
1. ไปที่เมนู "ครุภัณฑ์"
2. ระบบจะแสดงรายการครุภัณฑ์ทั้งหมด
3. ใช้ช่องค้นหาเพื่อกรองข้อมูล
4. เลือกสถานที่เพื่อกรองตามสถานที่

#### เพิ่มครุภัณฑ์ใหม่
1. คลิกปุ่ม "เพิ่มครุภัณฑ์"
2. กรอกข้อมูลครุภัณฑ์:
   - **ประเภท**: เลือกประเภทครุภัณฑ์ (Computer, Printer, Server, etc.)
   - **Service Tag**: หมายเลขประจำครุภัณฑ์
   - **ผู้รับผิดชอบ**: เลือกผู้ใช้งานที่รับผิดชอบ
   - **สถานที่**: เลือกสถานที่ติดตั้ง
   - **วันที่รับ**: วันที่ได้รับครุภัณฑ์
   - **สถานะ**: สถานะการใช้งาน
3. คลิก "บันทึก"

#### แก้ไขครุภัณฑ์
1. คลิกไอคอน "แก้ไข" ในแถวของครุภัณฑ์
2. แก้ไขข้อมูลที่ต้องการ
3. คลิก "บันทึก"

#### ลบครุภัณฑ์
1. คลิกไอคอน "ลบ" ในแถวของครุภัณฑ์
2. ยืนยันการลบ

### 4. จัดการผู้ใช้งาน

#### ดูรายการผู้ใช้งาน
1. ไปที่เมนู "ผู้ใช้งาน"
2. ระบบจะแสดงรายการผู้ใช้งานทั้งหมด
3. ใช้ช่องค้นหาเพื่อกรองข้อมูล

#### เพิ่มผู้ใช้งานใหม่
1. คลิกปุ่ม "เพิ่มผู้ใช้งาน"
2. กรอกข้อมูลผู้ใช้งาน:
   - **Username**: ชื่อผู้ใช้งาน
   - **Password**: รหัสผ่าน
   - **Email**: อีเมล
   - **ชื่อ-นามสกุล**: ชื่อและนามสกุล
   - **ตำแหน่ง**: ตำแหน่งงาน
   - **สถานที่**: สถานที่ทำงาน
   - **บทบาท**: เลือก Admin หรือ User
3. คลิก "บันทึก"

### 5. จัดการสถานที่

#### ดูรายการสถานที่
1. ไปที่เมนู "สถานที่"
2. ระบบจะแสดงรายการสถานที่ทั้งหมด

#### เพิ่มสถานที่ใหม่
1. คลิกปุ่ม "เพิ่มสถานที่"
2. กรอกข้อมูล:
   - **ฝ่าย**: ชื่อฝ่าย
   - **แผนก**: ชื่อแผนก
3. คลิก "บันทึก"

### 6. รายงาน

#### ดูรายงานครุภัณฑ์
1. ไปที่เมนู "รายงาน"
2. เลือกประเภทรายงาน:
   - **สรุปครุภัณฑ์**: ภาพรวมครุภัณฑ์ทั้งหมด
   - **รายงานการบำรุงรักษา**: ประวัติการบำรุงรักษา
   - **รายงานตามสถานที่**: ครุภัณฑ์แยกตามสถานที่

#### ส่งออกรายงาน
1. เลือกรายงานที่ต้องการ
2. กำหนดช่วงวันที่ (ถ้ามี)
3. คลิกปุ่ม "ส่งออก Excel" หรือ "พิมพ์"

## การตั้งค่าระบบ

### 1. การตั้งค่าฐานข้อมูล
- แก้ไขไฟล์ `.env` เพื่อเปลี่ยนการตั้งค่าฐานข้อมูล
- เปลี่ยน `JWT_SECRET` สำหรับความปลอดภัย
- ตั้งค่า `CORS_ORIGIN` สำหรับการเข้าถึง

### 2. การตั้งค่าการแจ้งเตือน
- ตั้งค่าการแจ้งเตือนเมื่อครุภัณฑ์ชำรุด
- ตั้งค่าการแจ้งเตือนการบำรุงรักษา
- ตั้งค่าการแจ้งเตือนการหมดอายุ

### 3. การสำรองข้อมูล
- ตั้งค่าการสำรองข้อมูลอัตโนมัติ
- กำหนดความถี่ในการสำรอง
- ตั้งค่าสถานที่เก็บไฟล์สำรอง

## การแก้ไขปัญหา

### ปัญหาการบันทึกข้อมูล
1. ตรวจสอบการกรอกข้อมูลให้ครบ
2. ตรวจสอบการเชื่อมต่อฐานข้อมูล
3. ตรวจสอบสิทธิ์การเขียนข้อมูล
